AcceptanceTests_builtIn.js

Description:

This tests the built in functionality of GNU/Linux+GNOME, incl. magnifier,
onscreen keyboard, etc. It tests the system using four preference sets:
 (1) os_common: common terms compatible with linux built in settings
 (2) os_win7: application specific windows settings
 (3) os_gnome: application specific gnome settings
 (4) os_common2: this is only used to check when highContrast/enabled=false

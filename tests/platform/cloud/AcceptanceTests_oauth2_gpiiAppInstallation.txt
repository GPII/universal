AcceptanceTests_gpiiAppInstallation.js

Description:
This is a series of tests using the online flowmanager to test the workflow of the GPII app installations to request access tokens from GPII Cloud using OAuth2 resource owner GPII token grant.

Requirements:
Since it's using the online flowmanager, there are NO REQUIREMENTS to the system other than the GPII being installed (and able to run in local 'cloudBased' flowmanager mode).

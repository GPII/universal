This configuration file is used for testing the sst-chrome extension in Windows

It includes to the basic localInstall setup for acceptance tests (modifying the
preferences folder used), which in turn includes the standard development
config file (running GPII locally, using development setup).

This config sets the device reporter file to be 'chrome_sst.json', which will report
the Cloud4all SST-Chrome extension for Google Chrome as being installed on the system.
